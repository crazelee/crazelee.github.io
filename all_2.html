抽象工厂模式 - Abstract Factory

超级工厂模式，生产其他工厂，适合做个路由

抽象工厂类定义

<?php
 
class AbstractFactory
{
    public static function build($type='', $country)
    {
        if ($type == '') {
            throw new Exception("Invalid Type");
        }

        $className = "\\Api\\P{$country}\\{$type}";
        if (! class_exists($className)) {
            $className = "\\Lib\\$type\\General{$country}";
        }

        return new $className;
    }
}

结构

使用

// Creating new Sedan Car
$sedan = AbstractFactory::build('sedan', 'china');
 
// Creating new giant Bike
$suv = AbstractFactory::build('giant', 'germany');

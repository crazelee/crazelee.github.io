Linux 常用命令

Telnet

telnet localhost 23

  1. 可确定远程服务的状态
  2. 确定某个端口是否可用

sort

输出内容排序 sort -u file // 去重

uniq 去重

uniq -d 仅显示重复行
uniq -u 显示不重复行

xargs

读取输入数据重新格式化后输出

输出进程id ps -ef |awk '{print $2}' | xargs
